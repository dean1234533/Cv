# DB's Workouts Calculator

Public Vite/React app for the DB's Workouts Free Macro & Training Goal Calculator.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite.

## Build

```bash
npm run build
```

## Check Code

```bash
npm run lint
```

## Notes

The calculator stores optional local submissions in the visitor's browser storage. It does not require a backend service for users to calculate, view, share, or download results.
